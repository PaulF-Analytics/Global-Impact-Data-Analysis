# Global Impact Data Analysis
This repository contains a series of technical projects focused on socio-economic data and development economics.

## Tech Stack
- **Languages:** Python
- **Libraries:** Pandas, Statsmodels (OLS Regression), Matplotlib, Seaborn
- **Data Sources:** World Bank API, UNDP Datasets

## Project 1: Health Expenditure Efficiency
- **Focus:** World Bank API data analysis.
- **Goal:** Identifying how nations maximize life expectancy relative to health spending.

## Project 2: Nigeria & South Africa Econometric Comparison
- **Focus:** OLS Regression and HDI drivers.
- **Goal:** Analyzing industrial impacts on human development (1996–2023).
